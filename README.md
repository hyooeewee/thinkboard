## Introduction

## Stack

**Database**
* [MongoDB](https://www.mongodb.com/)
* Redis
  > on [Upstash](https://upstash.com/)

**Backend**
* Express
* Node.js

**Frontend**
* Vite
* React
* Tailwind CSS
* Daisy UI - UI Components

## Backend

### Init

```shell
cd backend && pnpm init
```

### Develop

```shell
pnpm i && pnpm run dev
```

## Frontend

### Init
```shell
cd frontend && pnpm create vite@latest .
```

### Develop
```shell
pnpm i && pnpm run dev
```
